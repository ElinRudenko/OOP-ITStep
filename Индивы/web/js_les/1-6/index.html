Modal Dialog

This is a modal dialog box that requires user attention. Modal dialogs are used when:

  • Important information needs to be confirmed
  • User input is required before proceeding
  • Critical warnings need to be acknowledged

Modal dialogs create focus by preventing interaction with the main application until the dialog is addressed.
